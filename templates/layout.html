CIRRUS Weather App: Climate Information, Research, and Records of the US

Description: This app gives historical statistical records of various climate conditions and weather parameters. The data can be displayed for either a specified month and year or a specified parameter. Currently, the data is for 1975-2021 and has been collected for all zipcodes of the USA from this weather site and has been populated in a database. If the user wants to see data for a specific month and year, they can select a month from a drop-down menu and input a year from 2011 to 2021 in the year input field. After the user presses the submit button, the program then searches the database, finds the following parameters and their corresponding zipcodes, and renders them on a map:

Maximum Temperature Minimum Temperature Average Temperature Wind Chill
Heat Index Precipitation Snow Depth Wind Speed
Wind Gust Visibility Cloud Cover Relative Humidity

For more information, hover over the marker, which will display a label with the city, state, and zipcode of that marker, as well as the parameter and the value of the parameter.

If the user wants to see data for a specific parameter, they can select a parameter from the drop-down menu. After the user presses the submit button, the program then searches the database, calculates the average for each year for that parameter, and displays the results on a chart. For more information on a specific data point, the user can hover over it and more data will be displayed.
